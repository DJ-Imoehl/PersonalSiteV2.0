    HexMap

    HexMap is a map generation software that creates a realistic map out of tiles that are hexagonal shaped. I love hexagon-based maps in video games, similar to the ones that Civilization builds. I get so much joy from just exploring the map, so I thought I'd build a website to generate these maps. I also wanted to learn React with this project, so I built this map generator with that library.
